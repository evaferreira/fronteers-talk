Hiding Content

Hidden from whom?

  • All users: Completely hidden
  • SR users: Semantically hidden
  • Sighted users: Visually hidden

Completely hidden


.element{
	display: none;
}
					

.element{
	visibility: hidden;
}
					

<div hidden>Element</div>

Semantically hidden


<div aria-hidden="true">Element</div>

Visually hidden


.visually-hidden { 
	display: block;
	height: 1px;
	width: 1px;
	overflow: hidden;
	clip: rect(1px 1px 1px 1px);
	clip: rect(1px, 1px, 1px, 1px);
	clip-path: inset(1px);
	white-space: nowrap;
	position: absolute;
}

Skip links, un-do Visually hidden

Use main tag with ID
