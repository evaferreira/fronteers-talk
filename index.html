Outlines

It's the keyboard user's cursor

Don't delete the outline

There are a lot of keyboard users.
You are probably one of them.

It doesn't have to be a snowman

Outline

Don't delete the outline

... Unless you replace it.


a:focus{
	outline: none;
	background-color: pink;
}

Outlines are cool, anyway!

  • They don't take any space
  • You can make them look awesome!
  • It's known to the user

2
Hiding Content

Hidden from whom?

  • All users: Completely hidden
  • SR users: Semantically hidden
  • Sighted users: Visually hidden

Completely hidden


.element{
	display: none;
}
					

.element{
	visibility: hidden;
}
					

<div hidden>Element</div>

Semantically hidden


<div aria-hidden="true">Element hidden from Screen Readers</div>

Visually hidden


.visually-hidden { 
	display: block;
	height: 1px;
	width: 1px;
	overflow: hidden;
	clip: rect(1px 1px 1px 1px);
	clip: rect(1px, 1px, 1px, 1px);
	clip-path: inset(1px);
	white-space: nowrap;
	position: absolute;
}

7
MOAR Media Queries

Light Level

  • Normal: Natural / artificial light
  • Dim: At night / low light
  • Washed: Sunlight, the vampire slayer

Light level


@media (light-level: dim) {
  p {
    background: #222;
    color: white;
  }
}

@media (light-level: washed) {
  p {
    background: white;
    color: black;
    font-size: 2em;
  }
}

This doesn't work anywhere (yet)

Inverted colors

  • None
  • Inverted

Inverted colors


@media (inverted-colors: inverted) {
  img{
	  filter: invert(100%);
  }
  input{
	  border: 2px solid skyblue;
  }
}

Currently works in Safari browser

High Contrast mode (Windows)

  • Active
  • Black on white
  • White on black

-ms-high-contrast


@media (-ms-high-contrast: active) {
	/* All high contrast styling rules */
}

-ms-high-contrast


@media (-ms-high-contrast: black-on-white) {
	/* Specific */
	.el { background-image: url(image2.png); }
}

@media (-ms-high-contrast: white-on-black) {
	/* Specific */
	.el { background-image: url(image3.png); }
}

Inverted colors Vs. High Contrast mode

  • Inverted colors doesn't fix a bad contrast
  • High Contrast mode doesn't invert images
  • High Contrast mode doesn't... invert colors.

@media (-ms-high-contrast: active) {
	body{
		/* This will be pink */
		background-color: pink;
	}
}
					

@media (inverted-colors: inverted) {
	body{
		/* This will NOT be pink */
		background-color: pink;
	}
}
